## 主题样式

请移步 [prismjs](https://prismjs.com/) 官网。

## 设置主题

```js
// .vuepress/config.js

module.exports = {
  theme: 'reco',
  themeConfig: {
    /**
     * support for
     * 'default'
     * 'coy'
     * 'dark'
     * 'funky'
     * 'okaidia'
     * 'solarizedlight'
     * 'tomorrow'
     * 'twilight'
     */
    codeTheme: 'tomorrow' // default 'tomorrow'
  }
}
```
